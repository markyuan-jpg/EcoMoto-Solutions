# Monitoring & Continuous Optimization Guide

## Weekly Tasks

### Every Monday: GSC Check

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Select property
3. Check:

| Report | What to Look For | Action |
|--------|------------------|--------|
| **Coverage** | Errors, excluded pages | Fix issues |
| **Performance** | Impressions, clicks | Note changes |
| **Links** | New links, lost links | Record data |

### Weekly: GA4 Review

1. Go to [analytics.google.com](https://analytics.google.com)
2. Check **Realtime** - any issues?
3. Check **Acquisition → All Traffic → Channels**:
   - Organic traffic trending?
4. Check **Engagement → Pages and screens**:
   - Top pages performing?
5. Check **Conversions → Events**:
   - generate_lead events tracking?

## Monthly Tasks

### Monthly: SEO Audit (Tool: Screaming Frog)

1. Download and run Screaming Frog
2. Spider: `https://www.ecomotosolutions.com`
3. Review:
   - 4XX errors (broken links)
   - Missing meta descriptions
   - Duplicate titles
   - Slow pages

### Monthly: SEO Report

Create a spreadsheet with:

## Quarterly Tasks

### Quarterly: Full Site Audit

1. Use [Site Audit](https://webmastertools.google.com) tool
2. Fix critical issues
3. Review Core Web Vitals
4. Update content if needed

### Quarterly: Content Refresh

1. Check product pages - specs up to date?
2. Check blog posts - links still work?
3. Add new case studies
4. Update pricing info (if public)

## When Things Go Wrong

| Problem | Cause | Fix |
|---------|-------|-----|
| Traffic dropped | Algorithm update | Wait 1-2 weeks, check GSC |
| Pages not indexed | robots.txt issue | Check and fix |
| Rankings lost | Content quality | Improve content |
| Manual action | Policy violation | Check GSC messages |
