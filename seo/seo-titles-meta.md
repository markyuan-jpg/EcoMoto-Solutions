## Title Best Practices

| Rule | Limit |
|------|-------|
| Title length | 50-60 characters |
| Include brand | At end (after pipe) |
| Include keyword | Primary keyword first |
| Unique per page | No duplicates |

## Meta Description Best Practices

| Rule | Limit |
|------|-------|
| Description length | 150-160 characters |
| Call to action | Include (e.g., "Contact us") |
| Keywords | Include 1-2 naturally |
| Unique per page | No duplicates |
